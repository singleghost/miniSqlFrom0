# Index模块

B+树索引,支持在unique属性上建立索引,目前仅支持单值索引

采用懒惰删除,指针数小于阈值时不进行叶节点的合并

叶节点上每个entry的数据结构为 

    ---------------
    | value | RID |
    ---------------
    
value为实际的数据值,RID为记录在表中的位置

内部节点的数据结构为

    -----------------------------------------------------
    | PagePtr | value | PagePtr | value | ... | PagePtr |
    -----------------------------------------------------
    
比value小的存放在左子树,大于等于value的存放在右子树

叶节点之间构成一个双向链表,方便进行IndexScan
